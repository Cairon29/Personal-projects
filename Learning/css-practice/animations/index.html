Excercise 4: Key frames

Keyframes are used to define the animation of an element. They are used to define the different states of an animation. For example, you can define the initial state of an elementn, the state when the animation is in progress and the final state of the animation.

In this first box we are changing its background color as we do with transitions. Notice that the animation will be played only once. This Happends because we are using the animation without telling it to repeat.

We represent the state of the animation through percentages. In this case the 0% will have a light red background color, the 50% a light blue background and the 100% will have a light green background color. Once the animation is done, the element will return to its initial state (light red)
