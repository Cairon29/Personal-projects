CSS Animations


Excercise 1

In this exercise we are controlling the basic use of transitions on two different buttons. The first button will change to a different color and scale right after the mouse is hovered over it.

However, the second button will change same properties with a delay of 0.7secs.

Also, the stormtrooper will change its position and scale when the mouse is hovered over it. Notice that when user hovers over the image the stormtrooper does not go out of its box.
This happens because the box has the property of overflow: hidden.


Excercise 2

In this exercise we have a div with a background image. Our Darth Vader inside a box wihtout any effect is the image inside the div.

In the second image we have the same vader as a background image but this time with a linear gradient in its background

In the third image we have the same vader as a background image but this time with a moving effect that is infinite. This is working with the keyframes property.
Besides, when you hover vader it will turn bigger. Vader also has a background-blend-mode property that makes the background image and the background color to be mixed together. The property used in the background-blend-mode is darken which makes Vader image look different than the others.

In this exercises we are going to be using the ::before and ::after pseudo elements to create a simple animation for some HTML elements.

In this first exercise we can see an ASCII emoji wrapped in a "h3" tag which has a text in its left-side. The text is being injected right behind the h3 element using the ::before pseudo element. ::before places the content before the parent element itself.
